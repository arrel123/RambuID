<h1>RambuID - Panduan Penggunaan Aplikasi</h1>

RambuID merupakan aplikasi pengenalan rambu lalu lintas Indonesia yang dikembangkan untuk membantu pengguna dalam mengenali, memahami, dan mempelajari berbagai jenis rambu lalu lintas secara digital. Aplikasi ini dirancang untuk memberikan informasi rambu secara interaktif melalui perangkat mobile yang terhubung ke jaringan internet.

<h3>Cara Instalasi</h3>
Langkah-langkah untuk mengakses dan memasang aplikasi pada perangkat Android:

1. Unduh File: Akses URL http://151.243.222.93:56789/static/rambuid.apk melalui browser mobile untuk mengunduh file APK secara otomatis.

2. Pemasangan: Buka file rambuid.apk yang telah terunduh, lalu pilih tombol Install pada notifikasi instalasi yang muncul.

3. Inisialisasi: Saat pertama kali dijalankan, aplikasi menampilkan splash screen yang melakukan persiapan awal sistem dan pemuatan komponen antarmuka.

Autentikasi Pengguna
Aplikasi mewajibkan proses autentikasi agar pengguna dapat mengakses seluruh fitur yang tersedia:

1. Registrasi: Pengguna yang belum memiliki akun diwajibkan melakukan registrasi dengan mengisi data diri (Nama Lengkap, Email, Kata Sandi).

2. Login: Setelah berhasil registrasi, pengguna dapat masuk menggunakan email dan kata sandi yang telah dibuat.

<h3>Fitur Utama</h3>
1. Katalog & Edukasi Rambu

- Katalog Beranda: Menampilkan kategori rambu (Larangan, Peringatan, Perintah, Petunjuk) sebagai akses cepat menuju fitur edukasi.
- Pencarian: Pengguna dapat mencari rambu spesifik melalui kolom pencarian di menu edukasi.
- Detail Rambu: Menampilkan informasi mendalam meliputi nama, jenis, dan deskripsi fungsi rambu.
- Fitur Audio: Tersedia tombol Dengarkan untuk mendapatkan informasi rambu melalui suara.

2. Deteksi Rambu (Scanning)

- Metode Pemindaian: Pengguna dapat melakukan pengenalan rambu melalui kamera secara langsung atau memilih gambar dari galeri.
- Hasil Deteksi: Menampilkan identitas rambu yang terdeteksi beserta persentase akurasi dan deskripsi singkatnya.

3. Jelajahi (Peta Lokasi)

- Pemetaan: Menampilkan marker lokasi rambu-rambu lalu lintas yang tersebar di wilayah Batam.
- Informasi Lokasi: Pengguna dapat memilih marker tertentu untuk melihat informasi singkat mengenai rambu di lokasi tersebut.

4. Personal & Riwayat

- Manajemen Profil: Mengubah nama, email, alamat, kata sandi, serta foto profil di menu Tentang Pribadi.
- Bahasa: Mendukung pilihan Bahasa Indonesia (Default) dan Bahasa Inggris.
- Riwayat: Menyimpan daftar rambu yang pernah diakses atau dipindai sebagai referensi pembelajaran ulang.

<h3>Tech Stack (Analisis Sistem)</h3>

Berdasarkan fungsionalitas aplikasi, sistem ini mengintegrasikan:
- Mobile Platform: Android Application.
- Artificial Intelligence: Image Recognition untuk fitur deteksi rambu.
- Geospatial: Google Maps API atau sejenisnya untuk pemetaan wilayah Batam.
- Accessibility: Text-to-Speech (TTS) untuk edukasi berbasis suara.
